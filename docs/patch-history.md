# Historical Switchyard patch ledger

The original Switchyard patch queue was imported into this repository as one commit per patch. The identifiers below are retained as stable historical IDs and appear in commit messages as `Switchyard-Patch` trailers.

File paths and commands in individual test notes describe the repository layout at the time each patch was developed. Current development uses the commit history and the scripts under `switchyard/`; there is no second checked-in patch queue.

## Imported patches

  - Rationale: Chromium/CEF-based launchers, including Steam WebHelper, query process mitigation policy state during sandbox and renderer initialization. Returning success without initializing output buffers or remembering self-process policies can leave CEF with undefined policy state.
  - Scope: generic Wine `kernelbase` process mitigation API behavior. It does not inspect Steam process names and does not inject command-line switches.
  - Upstream status: Switchyard local compatibility patch, not submitted upstream yet.
  - Test notes: `./runtime/build/verify_wine_patch_queue.sh` verifies patch application and builds `dlls/kernelbase` when the Wine tree is configured. Set `WINE_BUILD_DIR` to point at an out-of-tree Wine build directory configured from this repository's `third_party/wine`.
  - Live validation still requires a built Switchyard Wine runtime plus a current official Steam install; passing patch apply-check is not a claim that every Steam build is fixed.
