# Security policy

Please report suspected vulnerabilities through GitHub's private vulnerability reporting for this repository. Do not open a public issue for credential exposure, sandbox escapes, arbitrary code execution, unsafe update behavior, or other issues that could put users at immediate risk.

Ordinary Wine compatibility regressions and launcher-specific rendering problems can use public issues when they contain no secrets, personal data, proprietary files, or exploit details.

Switchyard Wine does not provide or redistribute Apple Game Porting Toolkit components. Reports about obtaining, modifying, or bypassing licensing controls for third-party software are outside this project's scope.
